# mini-biblioteka
Cwiczenia 3  z przedmiotu PGO - prosta aplikacja biblioteczna w Javie.

## Opis
Aplikacja umozliwia zarzadzanie biblioteka: dodawanie ksiazek, wypozyczanie i zwracanie ich przez czytelnikow.

## Klasy
- **Ksiazka** - reprezentuje ksiazke (tytul, autor, liczba stron, dostepnosc)
- **Czytelnik** - reprezentuje uzytkownika biblioteki (imie, nazwisko, numer karty, liczba wypozyczen)
- **Biblioteka** - przechowuje kolekcje ksiazek, umozliwia wyszukiwanie, wypozyczanie i zwroty
- **Main** - scenariusz testowy aplikacji
